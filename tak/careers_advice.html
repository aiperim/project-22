POPULAR COMPUTER SCIENCE PATHWAYS

  • Game developer
    ... are responsible for designing and developing video games for PC, console, and mobile applications. Their job is to code the base engine, also may be involved in character design, level design, animation, and unit testing.
  • Software Engineer
    ... are responsible for applying the principles of software engineering to design, develop, maintain, test, and evaluate computer software.
  • Systems analyst
    ... are responsible for using analysis and design techniques to solve business problems using information technology, identify the organizational improvements, design systems to implement changes, and train others to use the systems.
  • IT consultant
    ... work in partnership with clients, advising them how to use information technology in order to meet their business objectives or overcome problems; and work to improve the structure and efficiency of IT systems in various organisations.
  • AI Engineer
    ... are responsible for developing new applications and systems that utilize AI to improve performance and efficiency, make better decisions, cut costs and increase profits.
  • Data scientist
    ... extract, analyse and interpret large amounts of data from a range of sources, using algorithmic, data mining, AI, machine learning and statistical tools, in order to make it accessible to businesses.
  • Cloud architect
    ... are responsible for overseeing a company's cloud computing strategy. This includes cloud adoption plans, cloud application design, and cloud management and monitoring.
  • Web developer
    ... are responsible for creating and maintaining websites, the site's technical aspects (its performance and capacity, which are measures of a website's speed and how much traffic the site can handle). May also create content for the site.
